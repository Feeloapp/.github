# Feelo

WhatsApp Business management platform for customer service teams.

Feelo gives teams a shared inbox for WhatsApp conversations with real-time collaboration, AI-powered automation, and multi-tenant organization isolation.

### Built with

**Frontend** Svelte + Tailwind | **Backend** Node.js + TypeScript + Express | **Database** PostgreSQL | **Infrastructure** AWS + Terraform

### Core capabilities

- Shared WhatsApp inbox with real-time messaging (Socket.IO)
- AI chatbot with RAG knowledge base (AWS Bedrock)
- Broadcast campaigns and template management
- Workflow automation builder
- Team chat (DMs and group conversations)
- Analytics dashboard and conversation summaries
- Mobile app (Android via Capacitor)
- Multi-tenant with per-organization data isolation
